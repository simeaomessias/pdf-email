PDF POR EMAIL


Olá, {{nome}}.

Este é um arquivo de teste. O e-mail {{email}} foi fornecido na página do projeto "PDF por E-mail".

A aplicação consiste em gerar um arquivo no formato pdf e fazer o envio através da conta sm-remetente@outlook.com.



TECNOLOGIAS UTILIZADAS

  • HTML5
  • CCS3
  • JavaScript
  • Node.js / Express.js / Handlebars.js
  • Html-pdf
  • Nodemailer

FUNCIONALIDADES

  • Um arquivo HTML (modelo deste documento) é compilado para uma função JavaScript com o Handlebars. Essa função recebe um objeto com os dados coletados no formulário e retorna uma string (HTML) com as propriedades do objeto inseridas.
  • Usando o Nodemailer (módulo Node.js para envio de e-mails), o arquivo é enviado a partir da conta sm-remetente@outlook.com
